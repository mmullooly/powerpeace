# Power & Peace

A React marketing site for **Power & Peace** — a turnless, real-time geopolitical simulation megagame for schools and groups.

Live at [powerpeace.org](https://powerpeace.org)

## Stack

- React 19 + Vite
- React Router v7
- Custom CSS (dark theme, glassmorphism)
- Deployed to GitHub Pages

## Getting started

```bash
npm install
npm run dev        # dev server at localhost:5173
npm run build      # production build
npm run preview    # preview the build locally
npm run deploy     # build + push to GitHub Pages
```
